## 🔧 Troubleshooting Resources

### Common Issues

**Sanity Connection Issues**
- https://www.sanity.io/docs/troubleshooting
- Check Project ID in `.env`
- Verify API tokens have permissions
- Restart dev server

**Clerk Authentication Issues**
- https://clerk.com/docs/support
- Verify keys in `.env`
- Check allowed URLs in Clerk dashboard
- Check browser console for errors

**Stripe Payment Issues**
- https://stripe.com/docs/testing
- Make sure using test keys
- Use proper test card numbers
- Check webhook configuration

**Deployment Issues**
- https://vercel.com/docs/support
- Check environment variables on platform
- Review build logs
- Clear cache and redeploy

## 💡 Pro Tips

1. **Keep documentation open** - Have multiple tabs open while coding
2. **Join communities** - Discord servers for Next.js, Sanity, Stripe
3. **Read error messages carefully** - They usually tell you what's wrong
4. **Use official tutorials first** - Before searching YouTube
5. **Test in development** - Before deploying to production
6. **Keep your .env safe** - Never commit it to GitHub
7. **Enable 2FA** - On all account logins (Sanity, Stripe, etc.)
8. **Use test environments** - Before going live
